# Design System for devcntr.app

## Overview
A professional, developer-focused design with a modern, high-tech aesthetic. Dark mode first.

## Color Palette

### Primary (Brand)
- **Primary**: `#3b82f6` (Blue 500) - Main actions, links.
- **Primary Hover**: `#2563eb` (Blue 600)
- **Secondary**: `#10b981` (Emerald 500) - Success, positive indicators.

### Neutrals (Dark Mode)
- **Background**: `#0f172a` (Slate 900) - Main background.
- **Surface**: `#1e293b` (Slate 800) - Cards, headers, footers.
- **Surface Hover**: `#334155` (Slate 700)
- **Text Primary**: `#f8fafc` (Slate 50) - Headings, main text.
- **Text Secondary**: `#94a3b8` (Slate 400) - Subtitles, descriptions.
- **Border**: `#334155` (Slate 700)

## Typography

### Font Family
- **Headings**: `Inter`, sans-serif
- **Body**: `Inter`, sans-serif
- **Code**: `JetBrains Mono`, monospace (if needed)

### Scale
- **H1**: 3rem (48px) - Hero Heading
- **H2**: 2.25rem (36px) - Section Headings
- **H3**: 1.5rem (24px) - Card Titles
- **Body**: 1rem (16px)
- **Small**: 0.875rem (14px)

## Components

### Buttons
- **Primary**: Solid background (Blue 500), White text, Rounded corners (0.5rem).
- **Secondary**: Transparent background, Border (Slate 700), White text.
- **Text**: Transparent background, brand color text.

### Cards
- **Background**: Surface (Slate 800)
- **Border**: 1px solid Slate 700
- **Hover**: Slight lift (transform: translateY), Border color change or Glow.

## spacing
- **Container**: Max-width 1200px, centered.
- **Section Padding**: 4rem vertical.
